┌───────────────────────────────────────────────────────────────────────────────┐
│ STEP 6: SETUP DATABASE                                          ⏱️  5 minutes │
└───────────────────────────────────────────────────────────────────────────────┘

  ☐ 6.1 Install Railway CLI locally
        Command: npm i -g @railway/cli

  ☐ 6.2 Login to Railway
        Command: railway login

  ☐ 6.3 Link to your project
        Command: railway link

  ☐ 6.4 Run database migrations
        Command: railway run python manage.py migrate

  ☐ 6.5 Create superuser account
        Command: railway run python manage.py createsuperuser
